How to use Kundali Darsan:
	-Entetr your name.
	-tap on iswi or Bikram date.
	    -selection calendar open.
    	-select date from calendar and click set.
	-or enter date/time manually in the same format shown.
	-Select birth country.	
	-Select birth place.
	-or search country/place using search button.
	-Tap save to save information.
	-Tap Prepare button to prepare kundali/Janm patrika.
	-Tap Delete icon to delete record.
	-Tap File Icon to load save kundali.

Features avialable:	
	-Add/Edit/Delete Country and place Latitude,Longitude Using Googlemap.
	-Basic forecast regarding info entered.
	-Planet/Graha status at that time.

	-Preparation of Kundali/Birth chart and other charts

		-Bhab kundali
		-Saptamansa kundali
		-Dwadasansa kundali
		-Drishyank kundali

		-Chandra/rasi kundali
